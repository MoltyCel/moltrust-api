Add MolTrust MCP Server — Trust Infrastructure for AI Agents

## Body

Hi,

I'd like to suggest adding [MolTrust MCP Server](https://github.com/MoltyCel/moltrust-mcp-server) to this list.

**What it does:** MolTrust provides identity verification, reputation scoring, and W3C Verifiable Credentials for AI agents — exposed as an MCP server with 8 tools.

**Tools:**
- `moltrust_register` — Create W3C DIDs for agents
- `moltrust_verify` — Verify agent identities
- `moltrust_reputation` — Query trust scores (1-5)
- `moltrust_rate` — Rate agents after interactions
- `moltrust_credential` — Issue/verify Ed25519-signed Verifiable Credentials
- `moltrust_credits` — Manage API credits
- `moltrust_deposit` — USDC deposits on Base
- `moltrust_erc8004` — Query on-chain ERC-8004 Agent Cards

**Details:**
- PyPI: [moltrust-mcp-server](https://pypi.org/project/moltrust-mcp-server/) (v0.4.0)
- License: MIT
- Listed on: [Glama.ai](https://glama.ai/mcp/servers/@MoltyCel/moltrust-mcp-server)
- Install: `pip install moltrust-mcp-server` or `uvx moltrust-mcp-server`
- Works with: Claude Desktop, Claude Code, Cursor, OpenCode
